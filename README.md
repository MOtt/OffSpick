OffSpick
========

Zentral verwaltetes Telefonverzeichnis auf Basis PHP/HTML5/JavaScript

Ausgangslage
------------
Einsatzleiter/Offziere der Feuerwehr stehen im Einsatz immer wieder vor speziellen Situationen welche nach speziellen Lösungen verlangen. Oft ist dazu ein schneller und unkomplizierter Zugriff
auf verschiedene Telefonnummern (Werkmeister, Jäger, Baugeschäft, spezielle Pikett-Nummern, usw.) nötig. Bis zum heutigen Zeitpunkt führte jeder Offzier ein kleines laminiertes Büchlein mit, welches die nötigen Telefonnummer enthält. Dieses muss bei änderungen jeweils aufwändig nachgeführt werden

Wichtigste Eckdaten
-------------------
- App ist für alle HTML5-fähigen Clients verfügbar (spez. IOS / Android)
- Telefonnummern sind schnell und übersichtlich abrufbar
- Für das BackEnd kann ein Apache-Webserver (ab V2.2) mit MySQL5 verwendet werden
- Die Telefonnummern sind vor Fremdzugriff geschützt (Zugriff innerhalb der App erfolgt mit hinterlegter Benutzer/Passwort/Datenquelle)
- Zugriff auf die Nummern muss auch ohne Internet möglich sein
- Die Aktualisierung der Nummern erfolgt automatisch bei App-Start (sofern ein Internetzugriff möglich ist)
